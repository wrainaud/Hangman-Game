Getting Started
---------------

The easiest way to get started is to clone the repository:

```bash
# Get the latest snapshot
git clone --depth=1 https://github.com/wrainaud/Rickman.git myproject

# Change directory
cd myproject

# Create Config file to hook into Firebase
touch config.js 
```
